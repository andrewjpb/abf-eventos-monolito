# CLAUDE.md - Instruções para o Claude Code

## Projeto

ABF Eventos Monolito - Sistema de eventos da ABF (Associação Brasileira de Franchising).
Stack: Next.js 15, Prisma, PostgreSQL, MinIO (S3), TypeScript.

## Commit

Após concluir alterações, **sempre criar um commit** com mensagem descritiva em português. **Nunca** incluir "Co-Authored-By", "Claude", "IA", "AI", "Generated by" ou qualquer referência a inteligência artificial na mensagem de commit.

## Deploy

Após concluir uma alteração solicitada pelo usuário e verificar que tudo está correto (TypeScript compila sem erros com `npx tsc --noEmit`), **sempre fazer o deploy em produção** executando:

```bash
npm run docker:deploy
```

Este comando faz build, push e remote update automaticamente.

## Estrutura importante

- **MinIO centralizado**: `src/lib/minio.ts` - Todos os uploads devem usar o cliente exportado deste módulo. Nunca criar instâncias duplicadas do cliente MinIO.
- **Banco de dados**: PostgreSQL via Prisma (`src/lib/prisma.ts`)
- **Autenticação**: Lucia Auth (`src/features/auth/`)
- **Server Actions**: Todas as actions ficam em `src/features/*/actions/`

## Idioma

O projeto é em português brasileiro (pt-BR). Mensagens de erro, logs e UI devem estar em português.
